########## ALGUNS PASSOS REALIZADOS NA AULA PARA TRABLHAR COM A CONTRIBUIÇÃO #########

1º PASSO: 
	-> Faça um FORK do repositório que deseja contribuir

2º PASSO:
	-> Faça um clone localmente utilizando o método HTTPS ou SSH do repositório

3º PASSO: Utilize o REMOTE UPSTREAM para manter seu repositório local atualizado

	-> "git remote add upstream <link_https_do_repositorio>"
	-> "git pull upstream main" = Para baixar e mesclar as alterações no seu repositório local com base na Branch "Main" desse 
	repositório em questão

4º PASSO: Crie uma nova Branch renomeando-a
	-> "$ git checkout -b feat/community/vitorjensen"
